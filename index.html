HTML Basics

Document Structure

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Page Title</title>
    </head>
    <body bg-color="relative-path" text="black" alink="red" vlink="purple" link="blue">
        <h1>Hello World</h1>
    </body>
</html>

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • <title>
  • <link rel="relationwithdoc" href="pathof" type="MIME">

Semantics for body

Text Elements

  • <br> - Line break
  • <h1> to <h6> - Headings
  • <p align="left|right|center"> - Paragraph
  • <strong> - for bold(used by senior devlopers) alternative : <b>
  • <em> - Emphasize
  • & nbsp; - Blanck space raw text element
  • <a href="url" target="_blank"> - Link(anchor) setter
  • <img src="image.jpg" alt="description" title="onhovertxt" align="l|r"> - Images getter

Code snippet Elements

  • <code> - code snippet
  • <pre> - preserve space
  • <var> - for code snippet variables
  • <samp> - comment and captions

Extra Elements

  • <details> and <summary> - for expand and collapse
  • <feildset> and <legend> - frame with head
  • <small> and <big>
  • <blockquote> - similar to paragraph but have left and right identation

Multimedia Elements

  • <marquee behavior="" direction="left/right/up/dowm" scrollamount="100" onmouseenter="stop()" onmouseout="start()"> - for moving
  • <audio> - for audio
  • <video src="" autoplay loop controls> - for video

Tables

CSS Selectors

  • element , .class , #id , element, element - Primary selectors
  • element childall , element > child , element + adjacent , element ~ adjacentall - Rational Selectors
  • [attribute] , [attribute="value"] , element[attribbute="value"] - attribute selector
  • :hover, :active, :focus - dynamic pseudo classes
  • :first-child , :last-child , :nth-child(levelnumber) , :nth-of-type(levelnumber[n]) - Structural pseudo classes
  • :enabled , :disabled , :read-only , :checked , :focus - element state pseudo classes
  • :valid , :invalid , :required , :optional - validation state pseudo classes
  • ::first-line , ::first-letter , ::before , ::after , ::placeholder , ::selection - behavioural & occurence classes
  • * - Universal selector
  • :lang(en) - language selector
  • :root - root selector

CSS Box Model

  • margin: top right bottom left;
  • border: width style color;
  • padding: top right bottom left;
  • width: value;
  • height: value;
  • background-color:rgba/hex/colorName/hsla;
  • box-shadow/text-shadow: horizontal vertical blur color

CSS display flex

Container Properties

  • display: flex;
  • flex-direction: row | column;
  • justify-content: flex-start | center | flex-end | space-between | space-around;
  • align-items: stretch | flex-start | center | flex-end | baseline;

Item Properties

  • flex-grow: number;
  • flex-shrink: number;
  • flex-basis: length | auto;
  • align-self: auto | flex-start | center | flex-end | baseline | stretch;

CSS display grid

Container Properties

  • display: grid;
  • grid-template-columns: 100px 1fr;
  • grid-template-rows: auto 100px 1fr;
  • gap: 10px;

Item Properties

  • grid-column: 1 / 3;
  • grid-row: ;
